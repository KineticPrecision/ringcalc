Inputs
First: Wheel information
Everything that spins that isn't the wheel: BS-8 arbor, Sopko adapter, balancing ring, screws. Weigh once, reuse.
Second: Use TTB Method and report
Read every hole at the 12-o'clock (top) position.
Let the wheel settle. The light side floats to the top; read the hole at 12 o'clock. This tells the tool which end of the axis is light — a rough read is fine.
Set the wheel rocking. At each reversal (the instant it stops and reverses), read the hole at 12 o'clock — one per side. These fix the balance axis precisely.
Pass target: period > 25 s.
